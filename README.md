# Project Manager

### Projeto de Avaliação

#### Repositório: https://github.com/marcus-santoss/project-manager.git

#### Documentação: [Aqui](apps/core/static/documentation.pdf)

### Rodar o projeto

#### Inicializar o ambiente

```shell
make up
```

#### Desligar o ambiente

```shell
make down
```

#### Reiniciar o ambiente

```shell
make restart
```
